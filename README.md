# Android应用管理器
Language: 中文简体 | [English](README-EN.md)

AAM是一款管理安卓应用的客户端，使用Flutter框架开发，支持安卓与桌面平台。

## 功能列表

- 支持多个平台:Android,macOS,Windows,Linux
- 批量操作
- 更全的应用信息获取
- 冻结/隐藏/卸载 应用
- 启动隐式意图
- 为 Activity 添加标签
- 备份与恢复应用(暂不支持aab应用)

## 注意！

**这个应用管理器需要 Root 权限！**

**在没有Root的设备上只支持部分功能，桌面端不受影响**
